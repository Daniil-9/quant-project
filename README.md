# Quant Project

##  Описание
Этот проект посвящён анализу финансовых данных и разработке простой торговой стратегии.

Цель — изучить работу с данными, базовую статистику и протестировать гипотезы на реальных рыночных данных.

---

##  Данные
Используются исторические данные по:
- Индексу Мосбиржи

Источник данных:
- https://ru.investing.com/indices/mcx-historical-data

---

##  Что сделано

### 1. Анализ данных
- загрузка и очистка данных  
- расчет доходностей  
- базовая статистика  

### 2. Фичи
- скользящие средние (moving average)  
- дополнительные признаки  

### 3. Стратегия
- реализована стратегия (например: momentum)  
- сигналы покупки/продажи
